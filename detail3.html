Memahami React Hooks

React Hooks memungkinkan functional components menggunakan state dan lifecycle methods tanpa menulis class components. Hooks mengubah cara kita membangun aplikasi React menjadi lebih sederhana dan reusable.

Hooks Paling Penting

useState

Mengelola state lokal dalam functional component.

useEffect

Handle side effects dan lifecycle methods.

useContext

Akses context tanpa Consumer wrapper.

useMemo

Optimasi performa dengan memoization.

Contoh useState

import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        Increment
      </button>
    </div>
  );
}

Custom Hooks

Custom hooks memungkinkan kita mengekstrak logic component menjadi fungsi yang reusable. Ini adalah salah satu fitur paling powerful dari React Hooks. 

function useLocalStorage(key, initialValue) {
  const [storedValue, setStoredValue] = useState(() => {
    try {
      const item = window.localStorage.getItem(key);
      return item ? JSON.parse(item) : initialValue;
    } catch (error) {
      return initialValue;
    }
  });

  const setValue = (value) => {
    try {
      setStoredValue(value);
      window.localStorage.setItem(key, JSON.stringify(value));
    } catch (error) {
      console.log(error);
    }
  };

  return [storedValue, setValue];
}

Best Practices

  • Selalu gunakan hooks di top level function
  • Gunakan useCallback untuk fungsi yang di-pass ke child
  • Pisahkan state yang tidak related
  • Buat custom hooks untuk logic yang complex
Perhatian: Jangan panggil hooks di dalam loops, conditions, atau nested functions. Hooks harus dipanggil di urutan yang sama setiap render.
