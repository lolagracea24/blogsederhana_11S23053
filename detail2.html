Menguasai CSS Grid

CSS Grid adalah sistem layout dua dimensi yang memungkinkan kita membuat layout kompleks dengan mudah. Berbeda dengan Flexbox yang satu dimensi, Grid memberikan kontrol penuh atas baris dan kolom.

Properti Dasar Grid

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-gap: 20px;
}

.item {
  grid-column: span 2;
  grid-row: span 1;
}

Tips Praktis

  • Gunakan `fr` unit untuk kolom yang fleksibel
  • Manfaatkan `minmax()` untuk responsive design
  • Kombinasikan dengan `auto-fit` dan `auto-fill`
  • Gunakan Grid Inspector di DevTools
Pro Tip: CSS Grid sempurna untuk layout halaman utama, sementara Flexbox ideal untuk komponen kecil seperti navbar atau card.
